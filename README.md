 ## Description
Welcome to ListenUp, the perfect platform for music lovers. This web application is centered on the idea of providing a socially interactive environment allowing users to connect to one another and share their opinions about musical artists and their creations.

By first creating and signing in to their own accounts, users are then able to find and create pages for their favorite artists, share their own opinions which gets displayed in their profile page and the home page, as well as get connected to other users by following each other and see their opinions on a multitude of artists. User opinions get saved and displayed on their personalized profile, allowing others to react to their posts as well as comment on them.

 ## Installation
 
  1. Clone the git repo

    ```bash
    git clone git@github.com:paulooliveira152012/ListenUp.git
    ```

  2. Install all dependencies in the project

    ```bash
    npm install
    ```

  3. Start both your react and mongoDB server (make sure you have concurrently installed on your machine) 

    ```bash 
    npm run develop
    ```

  4. Access your React server in your browser at localhost:3000
